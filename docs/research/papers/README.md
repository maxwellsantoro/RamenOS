# Research Papers

**Last Updated:** 2026-06-23
**Status:** Staging area

This directory tracks paper-sized research outputs when a doctrine-level problem
is large enough to deserve a serious written treatment.

Current candidates:

- `offers-airlock`: provider-authored offers, re-timing airlocks, observable
  contracts, and governed leakage.
- `org-kernel`: capability-governed AI organizations for large-scale open-source
  systems.
- `hil-evidence`: appliance-mediated HIL evidence without treating controller
  observations as target truth.
- `driver-factory`: Oracle traces, Reference Vaults, and native driver
  distillation.

Papers should remain connected to product slices and Foundry gates. A paper that
cannot name a product risk, claim boundary, and landing path belongs outside the
active research program.
